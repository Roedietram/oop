Een Ballonnen opdracht.
Dit is een complete script. Iets met Html (bootstrap template), php (OOP) en jquery.


Bekijk het script voordat je aanpassingen hierin gaat maken.


Opdrachten:
1. Maak het script werkend in je XAMPP of WAMP, LAMP

OOP aanpassing
2. Stel nu dat ik extra waardes in kleur en emoticons wil hebben? Hoe zou ik dit doen?

3. Ik wil een BallonGenerator class of een functie hebben? Het bestand waar de jquery naartoe gaat vind ik nog niet heel mooi, Ik wil bij het aanroepen van de ballon automatisch een ballon hebben of een random ballon verwerkt tot een class. Dus de logica hier vind ik niet op de juiste plek.
Eigenlijk moet je zelf nadenken hoe je dit zou uitwerken? Heb je een class nodig of een functie waarbij je dit zou kunnen oplossen. 

4. Ik wil aan de ballon een "happy thought" of "Happiness Quotes" aan meegeven.
Stel je voor je ziet een ballon met een tekst: "The Best Is Yet to Come", met een mooie opmaak.
Maak een plan en probeer het script aan te passen.
Bedenk wel in welke files de aanpassingen in moeten komen. Waarom is het belangrijk? Zo leer je dan zelf ook in te schatten hoeveel werk een aanpassing kan zijn.
Bedenk dat je in een template van Bootstrap zit. Hier kun je dus automatisch gebruik van maken zonder extra dingen voor te installeren.
