Extra opdracht (voor de toppers)

Kies minimaal één van de volgende uitbreidingen:




Optie 1 – Validatie

Zorg dat:

price niet negatief kan zijn

age niet onder 0 kan komen

Schrijf extra tests die controleren of foutieve invoer wordt tegengehouden




Optie 2 – Constructor

Voeg een __construct() toe aan één class

Zet alle properties via de constructor

Pas je tests aan




Optie 3 – Extra methode

Product: increaseStock(int $amount)

Animal: birthday() (verhoogt age met 1)

Person: moveTo(string $city)

Schrijf altijd eerst een test, daarna pas de code aan.
